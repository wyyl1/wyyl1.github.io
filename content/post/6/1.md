| 快捷键 | 功能 | 补充说明 |
| - | - | - |
| TAB / SHIFT + TAB | 选中后整段左右移动 | 多行一起移动 |
| Cmd + F2 | 全局替换文字 |  |
| Option + Shift + ↓ / ↑ | 复制当前行 |  |
| Cmd + Shift + K | 删除当前行 |  |
| Option + ↑ | 当前行向上移动 |  |
| Ctrl + ` | 切换终端 | Toggle Terminal |
